### 元素反应规则

好的。现在请尝试实现元素反应：

砖块与球的元素和.data中其颜色对应编号有关：

0，火，红色

1，水，深蓝色

2，风，蓝绿色

3，雷，紫色

4，冰，浅蓝色。



一个球和一个砖块碰撞时，与双方属性有关。

固定血量减一。

属性相同：无任何行为。


属性不同：

若一方为风：若球是风元素，扩散反应，把砖块邻近砖块的属性染成和碰撞砖块相同颜色。若砖块是风元素，把砖块邻近砖块的属性染成和球相同的颜色。



若是其它两种不同颜色（水火冰雷），则不区分球和砖块的顺序。


水+火：蒸发反应。砖块直接消失。球的位置和速度变为.data中定义的初始值。但是左方玩家不扣血。

水+冰：冻结。球停在碰撞的地方2秒钟不动，然后正常反弹。

水+雷：感电。场上所有水元素的方块血量-1。

火+冰：融化。场上所有冰元素方块变为水元素。

火+雷：超载。附近的砖块血量也减一。
